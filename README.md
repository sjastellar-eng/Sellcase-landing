# SellCase

AI-аналітика для продавців OLX.

SellCase допомагає визначити оптимальну ціну, відстежувати перегляди та прогнозувати швидкість продажу.

---

## Features

- AI аналіз ринку
- Рекомендована оптимальна ціна
- Відстеження переглядів у реальному часі
- Сповіщення про гарячі товари
- Прогноз продажу (2–3 дні, до 85% точності)

---

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- Vercel
